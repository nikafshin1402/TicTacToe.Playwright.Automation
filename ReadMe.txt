# TicTacToe Playwright Automation

This project is an end-to-end UI automation framework for a Tic-Tac-Toe web application.

## Tech Stack
- C#
- .NET
- NUnit
- Microsoft Playwright

## Project Structure
- Pages → Page Object Models
- Tests → UI Test Scenarios
- Core → Test base setup

## Covered Scenarios
- User registration and login
- Duplicate user validation
- Gameplay interaction with AI
- Profile update
- Game history validation
